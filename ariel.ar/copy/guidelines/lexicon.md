# lexicon.md — banco de palabras (ariel.ar)

Qué palabras usar y cuáles evitar al escribir copy de **ariel.ar**. Extiende
`../../brand/BRAND.md` §4 (Lenguaje) y §13 ("No"s tajantes). Es específico de
esta marca; otra marca tiene su propio lexicón. Cuando una decisión de copy
enseñe algo nuevo sobre una palabra, se agrega acá.

> Cómo leer este archivo: la columna **EVITAR** es dura — esas palabras no van.
> La columna **USAR** no es obligatoria, es la paleta de referencia.

## Verbos — voseo siempre

Formas correctas. Nunca usar la conjugación de tú (`construyes`, `puedes`).

| Usar | No usar |
|---|---|
| construís | construyes |
| podés | puedes |
| tenés | tienes |
| sabés | sabes |
| hacés | haces |
| empezás | empiezas |
| lanzás | lanzas |
| validás | validas |
| decidís | decides |
| elegís | eliges |

Verbos preferidos para CTAs y acción (concretos, de construcción):
**construí, lanzá, validá, publicá, empezá, creá, decidí, mostrá, probá.**

## EVITAR — tajante

Nunca escribir nada de esto (consolida BRAND §13):

- **Hype motivacional:** increíble · revolucionario · cambiará tu vida · el
  secreto · al fin · finalmente · épico · imperdible.
- **Promesas tech genéricas:** dominá · todas las herramientas · el mundo
  digital · el futuro es ahora · llevá tu X al siguiente nivel.
- **Adjetivos apilados:** innovador · disruptivo · transformador. Elegí cero.
- **CTAs vacíos:** sumate · descubrilo · animate · no te lo pierdas.
- **Registro hacker / Web3:** unlock · ship faster · 10x · matrix · cripto · web3.
- **Fantasía de ingresos:** hacete rico · ingresos pasivos · libertad financiera ·
  lifestyle · trabajá desde la playa.
- **Slang regional que achica:** vos sos un crack · posta · re copado.
- **Anglicismos innecesarios:** empoderarte · flujo de trabajo (usá *workflow*).

## USAR — términos de oficio (term-of-art) OK

Estos términos en inglés están bien porque son lenguaje real del oficio:
**MVP · brief · prompt · deploy · workflow · landing · feature.**

## Sustituciones — el reflejo a entrenar

| En vez de… | Escribí… |
|---|---|
| "increíble" / "alucinante" | la cosa concreta que es buena |
| "en tiempo récord" | el tiempo real ("en 6 semanas") |
| "todas las herramientas" | la herramienta concreta ("una sola herramienta") |
| "sumate" / "descubrilo" (CTA) | "empezá el módulo" · "ver sílabo" · "inscribirme" |
| "el mundo digital" | qué se construye ("tu primer producto digital") |
| "te enseño a triunfar" | "construís X" — el entregable concreto |
