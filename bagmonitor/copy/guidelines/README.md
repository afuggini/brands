# bagmonitor — copy guidelines

Copy guidelines for **BagMonitor**. This brand does not inherit from ariel.ar. It may use generic structures from `../../../shared/copy/` where useful, but BagMonitor's own rules always win.

All copy is in **English**.

## What lives here

- `lexicon.md` — approved terms, avoided terms, domain vocabulary
- `structures.md` — message structures for BagMonitor (hero formula, feature description formula, etc.)
- `formats/` — format-specific scaffolds and overrides

## The brand source of truth

Voice, positioning, ICPs, pillars, and hard no's live in `../../brand/BRAND.md`.
Visual system lives in `../../brand/DESIGN.md`.

These guidelines add operational copy rules on top of what's in BRAND.md. In case of conflict, BRAND.md wins.
