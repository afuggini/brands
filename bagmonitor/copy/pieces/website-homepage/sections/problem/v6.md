# The Problem — v6

**Eyebrow:** The Problem

**Heading:** Lost baggage is a visibility failure.

**Subheading:**
Airlines recover most delayed bags. The cost of that recovery — per event, per carrier, per year — is the problem. Every dollar spent on recovery is a dollar that visibility infrastructure would have prevented.
