# Research — BagMonitor

Esta carpeta contiene resúmenes y extractos de fuentes externas públicas relevantes para BagMonitor: videos de conferencias, entrevistas publicadas, artículos de industria y reportes de terceros.

**Diferencia clave con `../transcripts/`:** los transcripts son inteligencia primaria —llamadas y reuniones propias del equipo—. Los archivos aquí son fuentes secundarias, producidas y publicadas por terceros.

## Criterio de archivo

- Se documenta la fuente original (URL cuando está disponible).
- El contenido es un resumen fiel, no una transcripción literal.
- Se preservan datos numéricos, nombres propios y citas textuales relevantes.
- El idioma del archivo sigue el idioma original de la fuente.
