## Criterio de transcripción

- Transcripción limpia pero fiel al contenido original: se eliminan muletillas y ruido de audio, se conserva toda la información relevante.
- Los hablantes se identifican por nombre cuando es posible, o por rol cuando no se dispone del nombre.
- Las notas de análisis al final de cada archivo son un resumen editorial, no parte de la transcripción.
